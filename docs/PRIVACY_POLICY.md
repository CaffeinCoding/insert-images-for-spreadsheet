# 개인정보처리방침 (Privacy Policy)

**최종 수정일**: 2025년 11월 10일

## 1. 개요

"Insert Images For Spreadsheet" (이하 "본 서비스")는 사용자의 개인정보 보호를 최우선으로 생각합니다. 본 개인정보처리방침은 본 서비스가 어떤 정보를 수집하고 어떻게 사용하는지 설명합니다.

## 2. 핵심 원칙: 데이터 수집 없음

**본 서비스는 사용자의 개인정보를 수집, 저장, 전송하지 않습니다.**

## 3. 접근 권한 및 사용 목적

본 서비스는 다음과 같은 권한을 요청하며, 각 권한은 명시된 목적으로만 사용됩니다:

### 3.1 현재 스프레드시트 접근 권한

- **권한**: `https://www.googleapis.com/auth/spreadsheets.currentonly`
- **목적**:
  - 사용자가 선택한 셀의 위치와 크기를 읽기
  - 사용자가 선택한 이미지를 스프레드시트에 삽입
  - 이미지 삽입 취소(Undo) 기능 제공
- **범위**: 현재 열려 있는 스프레드시트 파일에만 접근하며, 다른 스프레드시트나 Google Drive 파일에는 접근하지 않습니다.

### 3.2 UI 표시 권한

- **권한**: `https://www.googleapis.com/auth/script.container.ui`
- **목적**: 사이드바와 메뉴를 표시하기 위한 권한입니다.

### 3.3 부가기능 실행 권한

- **권한**: `https://www.googleapis.com/auth/script.addons`
- **목적**: Google Workspace 부가기능으로 실행되기 위한 기본 권한입니다.

## 4. 데이터 처리 방식

### 4.1 이미지 파일

- **로컬 처리**: 사용자가 업로드한 이미지는 브라우저 내에서만 처리되며, 외부 서버로 전송되지 않습니다.
- **임시 저장**: 이미지 데이터는 사용자가 "적용" 버튼을 클릭하여 스프레드시트에 삽입할 때까지 브라우저 메모리에만 임시로 존재합니다.
- **자동 삭제**: 사용자가 "저장" 버튼을 클릭하거나 사이드바를 닫으면 모든 이미지 데이터가 메모리에서 삭제됩니다.
- **서버 저장 없음**: 본 서비스는 사용자 데이터를 저장할 서버 인프라가 없습니다.

### 4.2 설정값 저장

- **브라우저 로컬 저장소**: 사용자의 편의를 위해 다음 설정값이 브라우저의 localStorage에 저장됩니다:
  - 행/열 개수
  - 행/열 간격
  - 이미지 크기 설정
  - 비활성 셀 정보
- **저장 위치**: 이 데이터는 사용자의 브라우저에만 저장되며, 외부로 전송되거나 서버에 저장되지 않습니다.
- **삭제 방법**: 사용자는 브라우저의 쿠키 및 사이트 데이터 삭제 기능을 통해 언제든지 이 데이터를 삭제할 수 있습니다.
- **기기 간 동기화 없음**: 설정값은 기기나 브라우저 간에 동기화되지 않습니다.

### 4.3 저장되지 않는 데이터

본 서비스는 다음 데이터를 저장하거나 수집하지 않습니다:

- 이미지 파일 및 이미지 데이터
- 스프레드시트 내용
- 사용자의 Google 계정 정보
- 사용 기록이나 분석 데이터
- 사용자 행동 또는 추적 데이터
- 개인 식별 정보(PII)

## 5. 제3자 데이터 공유

본 서비스는 어떠한 데이터도 제3자와 공유하지 않습니다. 분석 서비스, 광고 네트워크 또는 기타 제3자 서비스는 본 서비스에 통합되어 있지 않습니다.

## 6. Google과의 데이터 공유

본 서비스는 Google에 다음 정보만을 전송합니다:

- 사용자가 "적용" 버튼을 클릭했을 때, 선택한 이미지를 스프레드시트에 삽입하기 위한 API 호출

이는 Google Apps Script의 정상적인 작동을 위해 필요한 최소한의 통신이며, 개인식별정보는 포함되지 않습니다.

## 7. 쿠키 및 추적 기술

본 서비스는 다음을 사용하지 않습니다:

- 쿠키
- 분석 도구 (Google Analytics 등)
- 광고 추적
- 사용자 행동 추적
- 추적 픽셀 또는 비콘

## 8. 아동의 개인정보 보호

본 서비스는 의도적으로 만 13세 미만 아동의 개인정보를 수집하지 않습니다. 본 서비스는 어떠한 개인정보도 수집하지 않으므로, 모든 연령대가 안전하게 사용할 수 있습니다.

## 9. 데이터 보안

- 본 서비스는 사용자 데이터를 서버에 저장하지 않으므로, 서버 해킹이나 데이터 유출의 위험이 없습니다.
- 모든 데이터 처리는 사용자의 브라우저와 Google의 보안 시스템 내에서 이루어집니다.
- 사용자는 자신의 기기 및 브라우저 환경의 보안에 대한 책임이 있습니다.

## 10. 사용자 권리

사용자는 다음과 같은 권리를 가집니다:

- 언제든지 본 서비스의 사용을 중단할 권리
- 브라우저 설정을 통해 localStorage 데이터를 삭제할 권리
- 부가기능을 제거하여 모든 접근 권한을 철회할 권리
- 처리되는 데이터(있는 경우)에 대한 정보를 요청할 권리

## 11. 데이터 보유

- **이미지 데이터**: 보유하지 않음; 삽입 후 또는 사이드바를 닫을 때 즉시 삭제됨
- **설정 데이터**: 사용자가 수동으로 삭제할 때까지 브라우저 localStorage에 보유됨
- **서버 데이터 없음**: 본 서비스는 서버나 데이터베이스를 유지하지 않으므로, 서버 측 데이터 보유가 없습니다.

## 12. 국제적 데이터 전송

모든 데이터 처리가 사용자의 브라우저에서 로컬로 이루어지므로, 국제적 데이터 전송이 없습니다. 유일한 통신은 Google Workspace의 정상적인 작동의 일부로서 Google 서버와의 통신입니다.
